# QuickQuiz

A real-time multiplayer quiz web app (Kahoot-style) with live lobby chat. Hosts create quiz sets and rooms; players join via room code as guests and interact via chat.

## Commands

- **Dev server**: `npm run dev` (runs on http://localhost:3000)
- **Build**: `npm run build`
- **Production server**: `npm run start`
- **Lint**: `npm run lint`
